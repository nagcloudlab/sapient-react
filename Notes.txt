1- what is component ?
--------------------------------------------------------------

    => UI fragment 

        e.g board,clock


    UI-component principles
    -----------------------------

    #1. Must have single-responsibility 
    #2. Must be loosely coupled from other  
    #3. Should be customizable via props/attributes
    #4. Can be stateless or stateful
    #5. On state change, always re-render
    #6. Communication between components 
        parent -> child ( via props )
        child -> parent ( event / function-call )
